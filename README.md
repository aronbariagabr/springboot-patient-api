# Spring Boot Patient API

## 📊 Overview
Demo REST API for patient records using Spring Boot. Showcases secure backend design and healthcare domain expertise.

## 🛠️ Tech Stack
- Java
- Spring Boot
- RESTful APIs

## 🚀 Features
- Add patient records
- Retrieve patient records by ID
- Simple in-memory storage for demo purposes

Highlights backend development skills and connects to my experience building HIPAA-compliant healthcare systems at CVS Health.
